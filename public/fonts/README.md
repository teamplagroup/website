# Fonts

Drop licensed **Söhne** woff2 files here:

- `soehne-buch.woff2` (weight 400)
- `soehne-kraftig.woff2` (weight 600)

Söhne is a paid typeface from **Klim Type Foundry**
(<https://klim.co.nz/retail-fonts/soehne/>). It must be licensed before it
can be self-hosted here — no font files are committed to this repo.

Until these files are dropped in, the site renders in the fallback stack
declared in `src/styles/global.css` (`Inter, system-ui, -apple-system,
"Segoe UI", sans-serif`).
